#### Pré Requisitos de Instalação
**[Node.Js](https://nodejs.org/en/download/package-manager/)**

### Instalação
>npm install

## Execução
>npm start

### Compilação do Executavel

* **Windows x32:**
     >npm run win32 
* **Windows x64:**
    >npm run win64

* **Linux x32:**
    >npm run linux32

* **Linux x64:**
    >npm run linux64
